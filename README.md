# ⚽ FIFA World Cup 2026 Player Performance Analysis

An end-to-end Exploratory Data Analysis (EDA) project that analyzes FIFA World Cup 2026 player performance data to uncover insights into player, team, and match performance using Python.

---

## 📌 Project Overview

This project explores a comprehensive FIFA World Cup player performance dataset containing **54,600+ player-match records** and **75 performance-related features**.

The objective is to transform raw football statistics into meaningful insights through data cleaning, visualization, and exploratory data analysis.

---

## 📂 Dataset Information

- **Rows:** 54,600+
- **Columns:** 75
- **Dataset Type:** Football Player Performance
- **Source:** Kaggle

---

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
- Kaggle

---

## 📊 Analysis Performed

- Data Quality Assessment
- Player Demographics Analysis
- Match Analysis
- Offensive Performance Analysis
- Passing Performance Analysis
- Defensive Performance Analysis
- Performance Metrics Analysis
- Team Performance Analysis
- Correlation Analysis
- Business Insights & Recommendations

---

## 📈 Key Insights

- No missing values or duplicate records.
- Offensive and defensive metrics provide a balanced view of player performance.
- Passing accuracy alone does not guarantee attacking success.
- Teams exhibit distinct tactical patterns through performance metrics.
- Correlation analysis highlights strong relationships among several performance indicators.
